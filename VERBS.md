## 使用说明

这些动词会在 Claude 处理任务时随机显示在 spinner 中，例如：

```
⠙ 光合作用中...
⠹ 炼丹中...
⠸ 摸鱼中...
⠼ 重构中...
⠴ 推送中...
```

## 自定义动词

如果你想添加自己的动词，可以编辑 `verbs/zh-CN.json` 文件：

```json
{
  "mode": "replace",
  "verbs": [
    "你的动词1",
    "你的动词2",
    ...
  ]
}
```

然后重新运行 `./install.sh` 即可。
